Ce travail est divisé en trois exercices. Pour réaliser
Pour chacun d'entre eux, vous devez bascimanete exécuter les commandes shell suivantes :

make exo{x}
./exo{x}/exo{x}.exe

où {x} peut être trois valeurs : 1, 2 ou 3. Pour chaque année.
Le dossier CR est un document auxiliaire permettant de comprendre le déroulement des travaux.
